# 🛒 Local Shop Order Management System

A full-stack web application that allows users to browse products, place orders, and enables admins to manage orders efficiently.

---

## 🚀 Features

### 👤 User Side
- View product catalog
- Add items to cart
- Place orders with details (name, phone, address)
- Simple and clean UI

### 🔐 Admin Side
- Admin login system
- View all orders
- Mark orders as **Delivered**
- Order status tracking (Pending / Delivered)

---

## 🛠️ Tech Stack

### Backend
- Java (Spring Boot)
- Spring Data JPA
- MySQL Database
- REST APIs

### Frontend
- HTML
- CSS
- JavaScript (Fetch API)
